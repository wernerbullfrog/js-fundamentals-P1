### The `if` Statement

- The `condition` must be "truthy" for the code inside of the `{}` to actually run.

```js
if (condition) {
  // do something
}
```

---

```js
let allegiance = '';
if (forceUseGood === true) {
  allegiance = 'jedi'
} else {
  allegiance = 'dark side'
}
```

---

```js
let number = 20;
if (number > 16) {
  console.log('Hold');
}
```

---

```js
let number = 20;
if (number > 16) {
  console.log('Hold');
} else {
  console.log('Hit me');
}
```

<img src={forLoop} />

---

#### Example

```js

  if (number % 2 === 0) {
    console.log(number);
  }
}

```
